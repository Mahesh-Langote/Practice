# Blog Application

This is a simple blog application built using Node.js and Express. Users can view blog posts, create new posts, and search for posts. The application uses EJS for templating and includes client-side JavaScript for dynamic interactions.

## Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine. You can download it from [Node.js website](https://nodejs.org/).

### Installation

 **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/blog-application.git
    ```

 **Navigate to the project directory:**

    ```bash
    cd blog-application
    ```

**Install dependencies:**

    ```bash
    npm install
    ```

### Running the Application

Start the server:

```bash
npm start
Visit [http://localhost:8080](http://localhost:8080) in your browser to access the blog application.

## Features

- View all blog posts on the home page.
- Click on a blog post to view its details.
- Create a new blog post by filling out the form.
- Optionally include an image URL or upload an image file.
- Search for blog posts using the search feature.

## Project Structure

- `app.js`: Main server file.
- `views/`: EJS templates for rendering HTML.
- `public/`: Static assets such as stylesheets and client-side scripts.
